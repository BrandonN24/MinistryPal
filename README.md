# MinistryPal
A collection of AI chatbot agents geared for ministry workflows.

# How to run frontend:
Run the following commands to start up the front-end locally.  
`cd frontend`  
`npm install`  
`npm run dev`

# How to run backend:
Run the following commands to start up the backend locally.
`cd backend`  
`pip install -r requirements.txt`  
`uvicorn app.main:app --reload`
